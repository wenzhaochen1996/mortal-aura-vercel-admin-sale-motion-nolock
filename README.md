# Mortal Aura

This package reads its default site content from:

- `public/mortal-aura-config.json`

## Deploy
1. Upload all files to GitHub
2. Import the repo into Vercel
3. Build command: `npm run build`
4. Output directory: `dist`

## Update published content
Replace `public/mortal-aura-config.json` with your latest exported JSON, then redeploy.
